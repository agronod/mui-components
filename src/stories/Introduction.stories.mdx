# Design System - Agronod

The design system is meant to help designers and developers alike

### Look and feel

We aim make UI's that conform to Agronod's look and feel across multiple applications

### Ease of use

We aim to make it as easy as possible to create UI's with high standards

### Builds on standards

The components are built on top of Google Material Design

- [Material Design](https://material.io/components?platform=web)
- [React Material UI](https://mui.com/)

<!-- ## Components

Components are interactive building blocks that combined makes the UI of an application. We follow Material Design's structure and naming as much as possible. -->

## Theme

The theme determines look and feel, it is by default Agronod but is customizable. Look for the Theme menu in the toolbar to view the components with desired theme applied.
